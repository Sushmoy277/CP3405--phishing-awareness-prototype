# CP3405--phishing-awareness-prototype
Scrum project prototype for phishing awareness training (lesson, quiz, dashboard)
## Team
- Scrum Master: Ornysha
- Proxy Product Owner: Ramiz
- Developer: Sushmoy

## Product Goal
Create a simple phishing-awareness training prototype that includes:
- A short phishing-awareness lesson
- A quiz to test understanding
- A basic dashboard for managers to see who completed the training

## Contents

- phising_awareness_lesson.txt  
  Text version of the phishing-awareness lesson used in the prototype.

- phising_awareness_quiz.txt  
  Link to the Google Form used as the phishing quiz.

- training_dash.txt  
  Link to the Google Sheet used as a simple dashboard to view quiz completions.
  
- user_stories.txt
  A short list of the user stories our team used in the sprint. These stories explain what the users need from the lesson, quiz, and dashboard.

- daily_scrums.txt
  Notes from our three Daily Scrum meetings. Shows what we did, what we planned next, and any blockers.

## Notes
This prototype is not a full production system. It is a simple, low-fidelity proof-of-concept used to support learning Scrum and demonstrating an increment at the end of the sprint
